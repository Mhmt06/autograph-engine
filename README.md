# AUTOGRAPH.ENGINE

A deterministic and non-deterministic finite state machine (FSM) simulation platform. This engine allows users to model complex state transitions, execute test strings, and validate formal grammar definitions directly within a reactive web interface.

## Tech Stack

* **Runtime & Framework:** Node.js, React (TypeScript)
* **Build Tooling:** Vite
* **Styling:** Tailwind CSS
* **Infrastructure:** GitHub (Vercel CI/CD)

## Production Deployment

The latest stable version is automatically built and deployed from the `main` branch:
[autograph-engine.vercel.app](https://autograph-engine.vercel.app)
